# Image Assets - Sourcing Guide

This folder holds all raster photography for the Resort Management System. The project follows an **editorial drone-photography direction** — cinematic wide shots of the Malaysian coast, warm golden-hour tones, and lifestyle-rich resort interiors. All images must be sourced from **CC0 / public-domain** providers (primarily Unsplash and Pexels) so the project remains fully redistributable.

## Recommended Sources

- **Unsplash** (https://unsplash.com) — preferred for hero photography and drone shots. High-resolution, editorial quality, fully CC0.
- **Pexels** (https://pexels.com) — fallback when Unsplash lacks coverage. CC0 license.
- **Pixabay** (https://pixabay.com) — wider variety for niche subjects (food, crafts, equipment). CC0 license.

## File Format Guidance

- Prefer **.jpg** for photos — smaller than PNG for full-bleed photography.
- Use **.webp** if smaller still and the project supports it (most modern browsers do).
- **Max width 1920px** for hero images.
- **Max width 1200px** for room/gallery cards.
- **800x800** for product squares (stall product photography).
- Compress with TinyPNG or Squoosh before committing — typical 40-60% size reduction with no visible quality loss.

## License

All images **MUST** be CC0, public domain, or have explicit commercial-use license. Always credit the photographer in a project-level `CREDITS.md` when possible — Unsplash provides the photographer name and a direct profile link on every photo page.

## Quick-Source Workflow

1. Open Unsplash (https://unsplash.com).
2. Search the suggested term from the table above.
3. Download highest resolution available.
4. Rename to match the filename in this guide.
5. Drop into `assets/img/`.
6. Page renders correctly on next reload.
